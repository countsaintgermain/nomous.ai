# Skill: UI & UX Validator
- **Stylizacja:** Wyłącznie Tailwind CSS. Używaj `clsx` i `tailwind-merge` do łączenia klas. Komponenty UI oparte na shadcn/ui.
- **Weryfikacja:** Po edycji UI użyj Browser MCP (Puppeteer), zrób screenshot na `localhost:3000` i upewnij się, że element wyświetla się poprawnie i jest responsywny.
- **Dostępność:** Dbaj o tagi semantyczne HTML, atrybuty `aria-*` oraz kontrasty kolorów.
